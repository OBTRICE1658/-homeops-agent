🧠 Product Brain (HomeOps.ai)

One central page to understand the full product — vision, structure, systems, and modules.

👁️ Vision

HomeOps.ai is the mental load operating system for high-performing families.

It acts as a household Chief of Staff — blending emotional intelligence, automation, and insight.

Core user: Dual-income parents managing school, schedules, logistics, and emotional labor.

🔧 Core Modules

Module Description Status
Chat Agent Emotional support + smart assistant with trained tone ✅ Functional, refining tone
Dashboard Coffee Cup View, load metrics, weekly priorities ⏳ Early build
This Week View Structured events pulled from chat/calendar ⏳ Partially implemented
Invisible Labor Extractor Pulls tasks, reminders, emotional flags from chat ✅ Basic prototype live
Calendar Pipeline Turns chat into events (via FullCalendar) ✅ Working, needs polish
Partner View Shared view for co-parenting & task split 🧠 Idea phase
Gmail Intelligence Surfaces important emails (school, brands, etc.) 🧠 Next up

🧠 Architecture Overview

  • Frontend: HTML/CSS/JS (public/ folder)
  • Backend: Node.js + Express (index.cjs)
  • Data: Firebase Firestore
  • Deployment: Render
  • Chat API: OpenAI via /chat endpoint
  • Calendar UI: FullCalendar JS → injects events dynamically
