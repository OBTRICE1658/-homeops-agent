📝 Required Google Cloud Console Setup

  1. Go to Google Cloud Console
  2. Select your project or create a new one
  3. Enable the following APIs:
    • Gmail API
    • Google Calendar API
    • Google+ API (for user profile)
  4. Go to Credentials → Create OAuth 2.0 Client ID
  5. Add these redirect URIs:
    • http://localhost:3000/oauth2callback
    • http://localhost:3000/auth/gmail/callback

🔍 Common Issues & Solutions

Error: redirect_uri_mismatch
Add http://localhost:3000/oauth2callback to your Google OAuth Client redirect URIs
Error: invalid_client
Check your GMAIL_CLIENT_ID and GMAIL_CLIENT_SECRET in .env file
Success Flow:
OAuth → Google Consent → Callback → Token Storage → Redirect to /scan or /calibrate
