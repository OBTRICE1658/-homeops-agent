Force Gmail Consent Screen

If Google is skipping the consent screen, try these methods to see the full authorization flow.

Note: If you've previously authorized HomeOps, Google may skip the consent screen to improve user experience. These methods force Google to show it again.
Method 1: Clear Browser Session
  • Open Chrome/Safari in Incognito/Private mode
  • Go to localhost:3000/onboard
  • Click "Connect Gmail Securely"
  • You should see the full Google consent screen
Method 2: Revoke App Permissions
  • Go to Google Account Permissions
  • Find "HomeOps" in your connected apps
  • Click "Remove Access"
  • Return to onboarding flow
